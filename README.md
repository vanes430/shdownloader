# shdownloader

Simple file downloader for Spigot and Velocity using Java's built-in libraries.

## Features
- Direct download from URL.
- Zero external dependencies (only Spigot/Velocity APIs).
- Multi-module Maven structure.

## Command
- `/shdown <url> <filename>` - Downloads a file to the plugin's data folder.
